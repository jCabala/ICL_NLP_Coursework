This repository contains code and outputs for the Natural Language Processing (NLP) coursework at Imperial College London.

## Files

- [BestModel.ipynb](BestModel.ipynb): Contains the model definition, training code, and local evaluation for the coursework. This notebook implements the final model used for experiments and includes training loops and evaluation metrics.
- [EDA.ipynb](EDA.ipynb): Contains all code, analysis, and visualizations produced during Exploratory Data Analysis (EDA). Use this notebook to view dataset inspection, feature exploration, and preprocessing experiments.
- [dev.txt](dev.txt): Contains predictions produced by the final model on the development set.
- [test.txt](test.txt): Contains predictions produced by the final model on the test set.

**Note**: All notebooks have been run in Google Colab.
